# finance-3-statement-model

Lifted and adapted from [`anthropic/financial-services`](https://github.com/anthropics/financial-services) under Apache-2.0. See `LICENSE-NOTES.md` for upstream attribution.

Activate when the user asks the agent to complete or populate a 3-statement financial model template (Income Statement + Balance Sheet + Cash Flow Statement) from an uploaded workbook.

Pairs with `finance-audit-xls` (verify the completed model) and `finance-statement-analysis` (read trends + ratios from the finished output).
