title: Agent Design
description: Build Fat-folder agents, sub-agents, and templates in ThinkWork.
---

ThinkWork agents are folders with memory, policy, skills, and routing. The root
folder describes the generalist. Subfolders describe specialists. The overlay
composer fills in anything a folder does not override, so small teams can start
with a single agent and grow into specialist trees without a migration.

Use this section when you are authoring templates, importing FOG or FITA-style
bundles, or deciding where a new skill belongs.

- [Folder Is the Agent](/agent-design/folder-is-the-agent/) explains the mental model.
- [Inheritance Rules](/agent-design/inheritance-rules/) shows how composed files resolve.
- [Authoring Templates](/agent-design/authoring-templates/) covers template design.
- [Import FOG/FITA Bundles](/agent-design/import-fog-fita/) covers normalized imports.
