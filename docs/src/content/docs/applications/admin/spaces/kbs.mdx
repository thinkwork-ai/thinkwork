Sources is the Space Studio tab for selecting which authored document sources are available inside a Space. Use it when a workroom needs access to specific documents, runbooks, policies, or customer material. The route remains `/spaces/:spaceId/kbs` for compatibility.

**Route:** `/spaces/:spaceId/kbs`

## What the tab controls

The Sources tab selects Space Brain Sources and any explicit legacy knowledge-base bindings for this Space. New retained documents should flow into Hindsight Space memory as Brain Sources; legacy Bedrock KBs are compatibility context.

Use the tab for:

- customer documents;
- support or policy runbooks;
- project documentation;
- department-specific source material.

Do not treat this tab as the full Memory surface. Space memory, Brain Sources, and legacy knowledge bases are related context sources with different contracts.

<Aside type="tip">
  If a document should inform only one customer or workflow, attach it through
  the Space that owns that work. If it should inform the whole tenant, manage it
  in Memory and link broadly where appropriate.
</Aside>
